# PwdGen 

A simple password generator\.

Type `-h` or `--help` for usage\.

```

______        ______   ____ _____ _   _
|  _ \ \      / |  _ \ / ___| ____| \ | |
| |_) \ \ /\ / /| | | | |  _|  _| |  \| |
|  __/ \ V  V / | |_| | |_| | |___| |\  |
|_|     \_/\_/  |____/ \____|_____|_| \_|

Simple password generator, version 0.0.1, build 2021-09-28
  -allow-repeat
        allow repeated characters (default true)
  -copies int
        copies of password (default 1)
  -digits int
        number of digits in the password  (default 4)
  -length int
        length of password specified  (default 16)
  -no-clipboard
        do not copy to clipboard
  -no-upper
        excludes uppercase letters  (default true)
  -symbols int
        number of symbols in the password (default 6)
  -version
        print version and exit

```
